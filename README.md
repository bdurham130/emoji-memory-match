#  Emoji Memory Match

**Created:** July 2025  
**By:** Brandon Durham  
**Objective:** Flip cards to find matching emoji pairs. Win in the fewest turns!

---

##  How to Play
1. Enter your name and press **Start Game**.
2. Click cards to reveal emojis.
3. Match all the pairs!
4. Reset at any time to play again.
